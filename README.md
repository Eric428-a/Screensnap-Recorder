# ScreenSnap Recorder

A **modern web-based screen capture and recorder** that lets you record your screen with audio, preview recordings instantly, and download videos easily. Built with **HTML5, JavaScript, and CSS**, it runs entirely in the browser—no installation or server needed.

---

## Key Features

* Record your screen with audio effortlessly
* Instant playback in the browser
* Download recordings in `.webm` format
* Fully front-end; no server required
* Compatible with Chrome, Edge, and Firefox (Safari support limited)
* Lightweight and easy to use

---

## Live Demo

Run the app locally by opening the `ScreenSnapRecorder.html` file in your browser.

---

## How to Use

1. Click **Start Recording** to capture your screen.
2. Click **Stop Recording** to finish the session.
3. Preview your recording directly in the video player.
4. Automatically download your recording for offline use.

---

## Project Files

* `ScreenSnapRecorder.html` — main HTML file with embedded JS and basic CSS
* `styles.css` *(optional)* — custom styling
* `script.js` *(optional)* — separate JavaScript logic for cleaner code

---

## Requirements

* Modern browser (Chrome, Edge, Firefox)
* HTTPS if deploying online (required for screen capture API)

---

## Deployment on GitHub Pages

1. Create a new repository (e.g., `screensnap-recorder`).
2. Upload your HTML file (and optional CSS/JS files).
3. Go to **Settings → Pages**, set the source branch to `main` and folder to `/root`.
4. Access your live app at:

```
https://<username>.github.io/screensnap-recorder/ScreenSnapRecorder.html
```
